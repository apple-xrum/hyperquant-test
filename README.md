## Версии

- **Node.js**: v23.6.1
- **npm**: v11.1.0

## Установка

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите приложение в режиме разработки:
   ```bash
   npm run dev
   ```

## Использование

После запуска приложения вы можете открыть его в браузере по следующему URL:
```
http://localhost:5173
```
